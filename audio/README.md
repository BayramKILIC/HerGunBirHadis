# Dossier Audio

Ce dossier contient les fichiers audio MP3 pour les 30 hadis.

## Instructions

Placez vos fichiers audio dans ce dossier avec les noms suivants :

- hadis_01.mp3
- hadis_02.mp3
- hadis_03.mp3
- ...
- hadis_30.mp3

## Format recommandé

- **Format** : MP3
- **Qualité** : 128-192 kbps
- **Durée** : Variable selon le hadis
- **Langue** : Arabe (lecture du hadis)

## Note importante

Sans les fichiers audio, le bouton de lecture sera visible mais ne jouera rien. Assurez-vous d'ajouter tous les fichiers audio correspondant aux hadis dans votre base de données.

## Exemple de structure

```
audio/
├── hadis_01.mp3  (Premier hadis)
├── hadis_02.mp3  (Deuxième hadis)
├── hadis_03.mp3  (Troisième hadis)
└── ... (jusqu'à hadis_30.mp3)
```
